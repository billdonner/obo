# obo

Flashcard app for kids with bundled sample decks.

## Sample Decks
Sample text decks live in `obo/SampleDecks/` and are bundled with the app.

Format:
```
Title: <Deck Title>

Q: <Question> | A: <Answer>
```

## Loading Order
1. `Documents/decks.txt` (if present)
2. Bundled sample decks
3. Built-in fallback samples
